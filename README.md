# 機械学習Ⅰ

> 続編: [機械学習Ⅱ](https://github.com/hgphys/ml2)

---

## 授業概要

１、２年次のプログラミングと情報処理に関する学習内容を結びつけ、機械学習の概念とその数学的基礎、及び具体的なアルゴリズムとその実装方法を身につける。具体的には、機械学習の概念を説明でき、Python によって教師あり学習の回帰・分類のアルゴリズムを表現できることが目標である。

## 到達目標

- 機械学習の概念と全体像を理解する
- 機械学習の数学的基礎を身につける（線形代数・微分・確率統計）
- 教師あり学習（回帰・分類）の具体的なアルゴリズムを Python で実装できる

## 教材構成

各回は以下の3点セットで構成されます。

1. **要点まとめスライド（PDF）** — 授業中の主たる説明資料。Google Drive で受講生に共有
2. **演習ノートブック (`ml1_XX.ipynb`)** — 要点を踏まえた問題文と、`%run` で実行する `.py` スクリプトの提示
3. **問題用 `.py` スクリプト (`ml1_XX_NN.py`)** — 学生はノートブックの指示に従ってスクリプトを編集し、実行結果から問題の答えを導く

## 環境構築（初回のみ）

### 1. Anaconda のインストール

以下のサイトを参考にインストールしてください。

- [Python 環境構築ガイド](https://www.python.jp/install/install.html)
- [Anaconda](https://www.anaconda.com/products/distribution)

> ユーザ名が日本語の場合、インストール先のパスに注意してください。

### 2. 仮想環境の作成

```bash
conda create -n ml_env python=3.11 -y
```

### 3. 仮想環境の有効化

```bash
conda activate ml_env
```

プロンプトに `(ml_env)` と表示されれば OK です。

### 4. 必要パッケージのインストール

```bash
pip install -r requirements.txt
```

> この環境は「機械学習Ⅱ」でも共通で使用します。

## 課題の取り組み方

各回のフォルダには以下のファイルが含まれます。

```
ml1_XX/
├── ml1_XX.ipynb       # 課題本体（Jupyter Notebook）
├── ml1_XX_01.py       # 問題1用スクリプト
├── ml1_XX_02.py       # 問題2用スクリプト
├── ml1_XX_03.py       # 問題3用スクリプト
└── data/              # データ配置用フォルダ（ある場合）
```

### 手順

1. **仮想環境を有効化する**

    ```bash
    conda activate ml_env
    ```

2. **JupyterLab を起動する**

    ```bash
    jupyter lab
    ```

3. **課題に取り組む** — ノートブック `ml1_XX.ipynb` を開き、指示に従って `.py` ファイルを編集します。ノートブック内で以下のように実行できます。

    ```python
    %run ml1_XX_01.py
    ```

4. **回答を提出する** — Moodle コースの小テストに回答します。

5. **作業を終えたら無効化する**

    ```bash
    conda deactivate
    ```

## 仮想環境の削除（任意）

すべての授業が終了し環境が不要になった場合は、以下で削除できます。

```bash
conda remove --name ml_env --all
```
